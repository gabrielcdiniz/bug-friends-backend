## Description

## Requirements

run the following script for give at Husky the need permissions for execute the tasks

```bash
chmod a+x -R .husky/
```

## Installation

```bash
yarn
```

## Running the Application

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Testing the Application

```bash
# unit tests
$ yarn test

# tests in watch mode
$ yarn test:dev

# test coverage with report html
$ yarn test:ui
```
